CI / CD Pipelines

What is continuous integration (CI) and why is it necessary?

... common code base (VCS)

... permanently checking in code

... triggering automated builds

What is continuous deploy (CD) and why is it necessary?

... always runnable

... enriching the current version with new features

... customer doesn't need to wait for next release

Travis CI .yml 

sudo: required

language: bash

services:
  - docker

before_install:
  - docker build -t trooper1989/react-uni -f ./App/app/Dockerfile.dev ./App/app

script:
  - docker run -e CI=true trooper1989/react-uni npm run test -- --coverage

before_deploy:
  - cd ./App/app && zip -r /$TRAVIS_BUILD_DIR/docker-app-$TRAVIS_BUILD_NUMBER.zip *

deploy:
  provider: elasticbeanstalk
  ...
